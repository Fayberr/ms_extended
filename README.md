# ms_extended

Utility functions for general Minescript usage and Hypixel specific scripting. Designed to reduce boilerplate and simplify automation, entity interaction, inventory access, and scoreboard parsing.

## Features

- Smooth, human-like rotation to exact yaw and pitch with easing and correction  
- Calculate yaw and pitch to face any entity or position  
- Get entities sorted by distance with full metadata  
- Read tablist entries directly from the client  
- Parse Hypixel SkyBlock scoreboard (area, speed, farming fortune, pet name/level, profile, bank, etc.)  
- Access hotbar items and get selected slot  
- Find hotbar items by Minecraft id or Hypixel id  
- Extract numeric values from formatted text safely  
- Lightweight, safe, and optimized for Minescript automation  

New features are constantly added.
